  • /discord/* - Discord API
  • /telegram/* - Telegram API

使用说明

所有请求将通过 Netlify Edge Functions 代理到目标服务器，同时保留原始请求的方法、头部和主体。

例如，访问 /hexo/post/hello-world 将会代理到 https://hexo-gally.vercel.app/post/hello-world。

由 Netlify Edge Functions 提供支持。
